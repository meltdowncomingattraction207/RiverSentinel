## 📊 What RiverSentinel Does

RiverSentinel helps people watch river conditions and respond to flood risk. It brings sensor data, alerts, and dashboards into one place.

### Main uses

- Track river levels from connected IoT sensors
- Review live and past water data
- View flood risk trends
- Send alerts to citizens and local authorities
- Check maps and status panels
- Follow changes across multiple river points

### Who it is for

- Local flood control teams
- Municipal staff
- Emergency response groups
- Environmental monitoring teams
- Community alert coordinators

## 🛠️ Troubleshooting

If RiverSentinel does not start, try these steps:

1. Make sure the download finished fully.
2. Extract the ZIP file if you used one.
3. Right-click the app and choose **Run as administrator**.
4. Check that Windows did not block the file.
5. Restart your computer and open the app again.
6. Download the latest release from the release page if the file looks old or damaged.

If the dashboard opens but shows no data:

1. Check your internet connection.
2. Confirm the sensor source is active.
3. Refresh the page or restart the app.
4. Make sure the correct site or region is selected.

If alerts do not appear:

1. Check the threshold settings.
2. Confirm notification access is enabled.
3. Make sure the sensor data is current.
4. Review the alert rules in the admin view.
